# 📊 AdventureWorks Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an **interactive Sales Dashboard built using Microsoft Power BI** to analyze business performance using the **AdventureWorks dataset**.
The dashboard provides insights into **revenue, profit, orders, product performance, and return rates** to help stakeholders make data-driven decisions.

The dataset was sourced from **Microsoft Excel** and transformed using **Power Query** before building visualizations in Power BI.

---

## 🎯 Business Objectives

The main goals of this dashboard are:

* Track overall **sales and profit performance**
* Monitor **order volume and return rates**
* Identify **top-performing products**
* Analyze **sales trends over time**
* Understand **product category performance**

---

## 📂 Dataset Information

* **Dataset:** AdventureWorks
* **Data Source:** Microsoft Excel
* **Domain:** Retail / Sales Analytics

The dataset includes information such as:

* Sales revenue
* Orders
* Product categories
* Product returns
* Order dates
* Product details

## 📈 Dashboard Features

### 1️⃣ Revenue Trend Analysis

* Weekly revenue visualization showing business growth over time.

### 2️⃣ Orders by Category

* Comparison of orders across product categories:

  * Accessories
  * Bikes
  * Clothing

### 3️⃣ Top 10 Products

* Displays the **best-selling products** based on:

  * Orders
  * Revenue
  * Return rate

* Monthly revenue
* Monthly orders
* Monthly product returns

### 5️⃣ Product Insights

* **Most Ordered Product Type:** Tires and Tubes
* **Most Returned Product Type:** Shorts
